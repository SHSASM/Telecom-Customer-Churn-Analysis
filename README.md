# Telecom-Customer-Churn-Analysis
Power BI dashboard analyzing 7,043 customers to identify churn drivers and revenue loss

Customer Churn Analysis Report:
Executive Summary
This project analyzes customer attrition for a Telecom company. By examining 7,043 customers, I identified a 26.54% churn rate, resulting in a monthly revenue loss of $137.09K.

Key Findings
The 1-Year Crisis: Retention drops significantly after the first year of service.

Contract Vulnerability: Month-to-Month contracts are the primary source of churn.

Competitor Impact: "Competitors" are the leading cause of churn, specifically in cities like San Diego and Los Angeles.

Technical Skills Demonstrated
Data Visualization: Created a 5-page interactive dashboard with consistent branding.

Advanced Features: Implemented page navigation, sync slicers, and dynamic titles.

Analysis: Performed Root Cause Analysis (RCA) using Tenure and Service distribution
